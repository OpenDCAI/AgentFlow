Overview Interface

Use this Overview to see what Agentflow is before you open pipelines or the skill hub. It is not a one-off task screen for a single agent—it is a unified data synthesis framework for heterogeneous agent environments. You will see how interactions between agents and their environments are organized across settings, and how those runs become trainable, evaluable data you can reuse downstream.

What you get here is bigger than any one prompt or tool: a repeatable synthesis workflow. Inside a single sandbox, you sample agent trajectories, capture tool use, environment feedback, and reasoning, then turn the highest-value paths into samples for training and evaluation. Agentflow is built so you can produce high-quality agent data consistently—not just check off a single ad-hoc task.

framework: unified agent data synthesis pipeline: 3-stage workflow sandbox: all-in-one environment scope: heterogeneous agent tasks output: training & evaluation data

What This Framework Is For

  • Get one unified workflow for data synthesis across agent environments, instead of a setup tied to just one task type.
  • Reuse the same framework across Web, RAG, MM-Doc, Deep Research, GUI, Text2SQL, Data Analysis, Embodied, and more.
  • Treat agent-environment interaction as first-class data, not just the final answer.

What Problems It Solves

  • Heterogeneous agent environments vary in inputs, operations, and feedback formats, so you need one abstraction layer to manage synthesis consistently.
  • High-quality agent data should include more than outcomes: trajectories, tool interactions, reasoning traces, and environment feedback.
  • A clear split between sampling, selection, and synthesis helps you separate exploration logs from final training-ready samples.

End-to-End Workflow

Stage 1
Trajectory Sampling

Your agent explores the environment step by step, uses tools, receives feedback, and builds candidate trajectories.

Stage 2
Trajectory Selection

Select the paths with higher information density and stronger training value, instead of keeping every raw exploration trace.

Stage 3
QA Synthesis

Convert selected high-value trajectories into grounded QA and other samples ready for training and evaluation.
