AgentFlow

The First Unified Agent Data Synthesis Framework for Custom Tasks with All-in-One Environment.

GitHub Models Datasets Docs

Overview

AgentFlow generates high-quality training and evaluation data across heterogeneous agent environments. It provides a unified, extensible paradigm for synthesizing agent trajectories, reasoning traces, tool interactions, and environment feedback.

RAGMM-DocDeep ResearchGUI Text2SQLData AnalysisEmbodied Agents

Key Features

Unified Synthesis

Synthesize complex agent training data with just a few lines of code across all environments.

All-in-One Sandbox

Built-in support for RAG, Doc, Web, Code, SQL, GUI, Embodied and more, easily extensible.

Model Consolidation

Jointly train a unified agentic model on mixed trajectories from all domains.

Novel Benchmarks

High-quality benchmarks for evaluating real-world agentic capabilities.

  • DeepResearch — Information Seeking Agent Consolidation
