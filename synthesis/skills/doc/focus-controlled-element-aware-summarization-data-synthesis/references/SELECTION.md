# Phase 2: Trajectory Selection Criteria
* **Acceptance Metrics:**
  * **Frame adherence must be complete.** Accept only if the trajectory covers all requested elements, aspects, or query needs that are supported by the source. This is the first selection gate for the skill. If the agent handles only the easiest parts of the frame, reject the example.
  * **Off-frame exclusion must be meaningful.** A high-quality instance should contain tempting but non-essential material that the summary correctly ignores. This proves the task is about control, not simply about summarization quality. If every major source detail belongs in the final answer, the example does not stress selective focus enough.
  * **Source support must be slot-specific.** For every included summary unit, the selector should be able to point to the frame item it serves and the evidence span that supports it. This prevents vague, globally relevant sentences from sneaking into a supposedly controlled summary. If a sentence cannot be justified under a slot, it should not survive.
  * **The final answer must remain concise under the frame.** Control should not collapse into a long extraction dump. Accept only when the summary compresses the on-frame material while remaining clearly organized around the requested lens. If the answer just enumerates raw evidence without synthesis, it is too weak.
* **Rejection Criteria:**
  * **Reject generic-summary instances.** If the final answer would still score well even without the control frame, the example is not distinctive enough for this skill. The frame must materially change content selection. Otherwise the task belongs to broader summarization.
  * **Reject under-specified frames.** Vague requests like “summarize the important parts” are not sufficient. The control lens must be explicit enough that two reviewers would largely agree on what belongs in or out. If the frame is ambiguous, reject the example before synthesis.
  * **Reject slot hallucination.** If a required element or aspect is missing in the source and the trajectory quietly invents one to complete the frame, discard the case. This is especially important in element-aware summarization where a missing slot can tempt the model to over-generalize. Honest partial coverage is better than fabricated completeness.
  * **Reject answer drift.** Discard any instance where the final summary begins on-frame but then drifts into background context, commentary, or extra story material. That drift is exactly what this skill is meant to train against. If a third of the answer is off-frame, it should fail selection.
