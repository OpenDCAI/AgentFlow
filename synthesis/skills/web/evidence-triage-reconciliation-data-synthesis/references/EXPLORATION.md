# Phase 1: Environment Exploration Guide
* **Exploration Strategy**:
  - Deliberately gather both supporting and contradictory evidence instead of stopping at the first answer-like hit. The agent should open multiple high-relevance results, compare their timestamps, and record whether each page is current, stale, or contextually misaligned. As an edge case, if a search snippet seems decisive but the underlying page weakens or qualifies the claim, the page-level content must override the snippet.
  - Normalize evidence along three axes before trusting it: relevance, freshness, and source role. A source can be topically relevant but still wrong for the asked comparison, or fresh but only tangentially related to the decision criterion. For example, a page about a company reaching one trillion can still be insufficient if the question asks for the most recent first-time crossing across companies.
  - Maintain an explicit conflict ledger while browsing. For each candidate answer, note which pages support it, which pages weaken it, and what unresolved comparison remains. This prevents the agent from collapsing the search too early when two candidates both appear plausible.
  - Use disambiguating retrieval moves that target the conflict itself. When two sources disagree, issue a follow-up search or navigate to a more authoritative page that directly resolves the disputed field such as date, version, event boundary, or ranking condition. A good edge case is querying the exact milestone phrase plus a date boundary to verify ordering rather than gathering more generic company pages.

* **Target Trajectory Profile**:
  - A target trajectory should include at least one tempting but wrong candidate. The false lead should look credible enough that a weak agent would likely stop there. If no serious distractor appears, the example does not sufficiently test triage.
  - The trajectory should make the conflict legible. There should be a point where two or more candidates remain plausible and the agent must explain what evidence would resolve the disagreement. Without that explicit decision point, the reasoning looks like ordinary retrieval rather than reconciliation.
  - A strong trajectory resolves the conflict using a criterion that is structurally tied to the question. The decisive step must compare the exact requested field, such as which date is later, which source is primary, or which page speaks to the asked entity rather than a nearby one. As an edge case, a final choice based only on prestige of source without matching the question’s comparison axis is inadequate.
  - The final answer should emerge only after at least one verification step against the leading candidate. The agent should not merely switch from one answer-like page to another answer-like page. It should show why the accepted evidence survives scrutiny and why the rejected evidence fails under the question’s exact requirements.
